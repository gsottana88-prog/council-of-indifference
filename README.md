# Council of Indifference

> Sei personaggi tragicomici analizzano il mondo così non devi farlo tu.

Un'app Next.js che interroga API pubbliche (Hacker News, NASA, National Weather
Service, GitHub Trending, Chuck Norris) e trasforma i dati in **6 punti di vista
tragicomici**. Nessuna API key necessaria. Solo template tragicomici e
cattiveria gratuita.

## Deploy in 1 click

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/council-of-indifference)

Oppure:

```bash
npm install
npm run dev     # sviluppo su http://localhost:3000
npm run build   # build di produzione
```

riconoscibili — come Curb Your Enthusiasm ma scritto in TypeScript.
