# Council of Indifference

> Six tragicomic personas analyze the world so you don't have to.

A Next.js app that queries public APIs (Hacker News, NASA, National Weather
Service, GitHub Trending, Chuck Norris) and transforms the data into **6
tragicomic perspectives**. No API keys needed. Just templates and free cynicism.

## Deploy in 1 click

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/council-of-indifference)

Or:

```bash
npm install
npm run dev     # local dev at http://localhost:3000
npm run build   # production build
```

## The Council

| Member | Source | Vibe |
|--------|--------|------|
| 🔥 The Startup Bro | Hacker News | Tech hype & disillusionment |
| 🌌 Cosmic Rick | NASA APOD | Interstellar nihilism |
| 🌦️ Weather Dad | National Weather Service | Weather & melancholy |
| 🐙 The Code Karen | GitHub Trending | Open source madness |
| 😐 Dad Joker | Chuck Norris API | Existential comedy |
| 🔮 The Oracle | The Cosmic Void | Useless prophecies |

## Philosophy

Pure templates. No LLM. No cost. Just recognizable tragicomic patterns —
like Curb Your Enthusiasm written in TypeScript.

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** (custom cosmic theme)
- **Public APIs** (HN Firebase, NASA, NWS, GitHub Trending, Chuck Norris)
- **Deploy**: Vercel (free) or any Node.js host
